# Excel & Sheets — Analysis Notes
*From IT Consultant Internship | Alankaeon Consulting | December 2025*

## How Excel Was Used During the Internship

### 1. Competitive Analysis Matrix
**Purpose:** Compare multiple tools/solutions for client recommendations

**Structure used:**
- Rows = Tools being compared (e.g., Tool A, Tool B, Tool C)
- Columns = Evaluation criteria (Price, Features, Ease of Use, Support, Scalability)
- Cells = Score (1-5) or actual data
- Final column = Weighted total score

**Key Excel features used:**
- Conditional formatting (color-coded scores)
- SUMPRODUCT for weighted scoring
- Freeze panes for easy scrolling
- Table formatting for clean presentation

---

### 2. Project Deliverable Tracker

**Purpose:** Track tasks, deadlines, and status across multiple deliverables

**Columns used:**
- Task Name
- Category (Research / Documentation / Presentation)
- Assigned To
- Due Date
- Status (Not Started / In Progress / Done)
- Notes

**Key Excel features used:**
- Data validation (dropdown for Status)
- Conditional formatting (Red = overdue, Green = done)
- Filters for sorting by status or category

---

### 3. Research Summary Sheet

**Purpose:** Organize raw research into clean, presentable data

**Approach:**
- Separate tabs for each research topic
- Summary tab pulling key points from each
- Charts generated directly from data for use in PowerPoint

---

## Key Excel Functions Used

| Function | Used For |
|---|---|
| `SUM` | Totaling scores, counts |
| `AVERAGE` | Average ratings across criteria |
| `IF` | Conditional labels (e.g., "High / Medium / Low") |
| `COUNTIF` | Counting items by category |
| `VLOOKUP` | Cross-referencing data across sheets |
| Conditional Formatting | Visual highlighting of key data |
| Charts (Bar, Pie) | Data visualization for presentations |
