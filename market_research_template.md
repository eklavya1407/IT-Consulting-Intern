# Market Research Template
## IT Tool / Solution Evaluation Framework

*Used during internship at Alankaeon Consulting Pvt. Ltd. | December 2025*

---

## Purpose

This template was used to conduct structured competitive research and evaluate technology tools for client recommendations. It ensures a consistent, objective comparison across multiple options.

---

## Research Framework

### Step 1: Define the Client Need
- What problem is the client trying to solve?
- What are the must-have features?
- What is the budget range?
- Who are the end users?

### Step 2: Identify Options
- List all relevant tools/platforms in the market
- Minimum 5-8 options for a comprehensive comparison
- Include both popular and niche solutions

### Step 3: Evaluation Criteria Matrix

| Criteria | Weight | Tool A | Tool B | Tool C |
|---|---|---|---|---|
| Cost / Pricing | 25% | | | |
| Features | 25% | | | |
| Ease of Use | 20% | | | |
| Scalability | 15% | | | |
| Support / Reviews | 15% | | | |
| **Total Score** | 100% | | | |

### Step 4: Summary & Recommendation
- Top 2-3 shortlisted options
- Recommended option with justification
- Risks and limitations to note
